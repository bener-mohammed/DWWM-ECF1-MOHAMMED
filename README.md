# Le Phosphore - ECF DWWM 2026

This project is a responsive website for Le Phosphore witch is a performance hall

# Technologies

- HTML5
- SASS / SCSS
- BEM methodology
- Vanilla JavaScript
- JSON
- Git and GitHub

# Pages

- Home page
- Programming page
- Practical information page

# Features

- Responsive navigation
- Curtain opening animation
- Dynamic show cards generated from JSON
- Filters by show type, date and availability
- Seat occupancy progress bar
- Accessibility-focused structure

# Installation

```bash
npm install
npm run sass
